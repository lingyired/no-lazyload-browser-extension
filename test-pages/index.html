🧪 Image Lazy Load Blocker 测试页面

1 原生懒加载 loading="lazy" Native 40张
HTML5 原生 loading="lazy" 属性，现代浏览器支持的标准懒加载方式
 2 data-src 模式 Data Attr 40张
最常见的懒加载实现：data-src、data-original、data-lazy-src 等属性
 3 lazysizes 风格 Library 40张
模拟 lazysizes 库：lazyload → lazyloading → lazyloaded 类名切换
 4 背景图片懒加载 Background 40个
背景图片懒加载：data-bg、data-background 属性
💡 使用说明：
  1. 先在浏览器中加载扩展
  2. 在扩展设置中添加 localhost（如果使用本地服务器）或对应域名
  3. 选择策略（技术拦截或自动滚动）
  4. 打开测试页面，验证图片是否立即加载
  5. 注意页面右上角的加载状态指示器
