Privacy Policy

Last Updated: May 6, 2026

Core Commitment: We do not collect any user data.

Data Collection

The "No lazyload - Disable Image Lazy Loading" browser extension (hereinafter referred to as "this extension") does not collect, store, or transmit any user personal data.

Local Storage

All settings and configurations of this extension are stored locally on your device using the browser's local storage API (chrome.storage.local / browser.storage.local), including:

  • The list of website domains you have added
  • Blocking strategy settings for each website
  • Auto-scroll parameters (speed, stay duration, etc.)
  • Custom lazy loading attributes configuration
  • Language preference settings

How Data Is Used

The above data is stored entirely on your local device and is used to:

  • Remember your configuration preferences for specific websites
  • Apply appropriate blocking strategies when visiting configured websites
  • Provide a personalized extension settings experience

Data Transmission

Important: This extension does not upload any data to remote servers or third-party services. All data remains in your local browser storage.

Permissions

The browser permissions requested by this extension are only used to implement core functionality:

  • storage: Used to save your settings locally
  • activeTab: Used to get the current website domain for adding configurations
  • scripting: Used to inject scripts into web pages to intercept lazy loading
  • <all_urls>: Used to run blocking functionality on all websites you configure

Third-Party Services

This extension does not contain any third-party tracking code, analytics tools, or advertising SDKs.

Data Security

Since all data is stored locally, data security is ensured by your browser and operating system. When you uninstall the extension, all related data will be automatically cleared.

Export/Import Feature

This extension provides configuration export/import functionality. The exported configuration file only contains setting data that you actively choose to export, which you manage and share yourself. This extension does not automatically export or upload any data.
